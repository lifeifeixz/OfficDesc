拼车规则

一、拼车费用计算

1. 根据上海市最新发布的《关于规范本市私人小客车合乘出行的实施意见》，我们按照合乘所发生的油费，车辆损耗，人力成本等进行合理的分摊。详情请在下单时参考计费规则说明。

二、拼车流程

1. 预约下单：乘客下单需至少提前20分钟；
2. 车主抢单：顺路车主收到乘客预约后，可以抢单；先到先得；
3. 乘客付款：乘客在车主抢单后，需要在指定时间内完成支付，逾期未支付，订单将自动取消；
4. 车主服务：车主根据订单需求接送乘客；
5. 车主收款：乘客点击“确认搭乘”后，车主将在次日12点后收到全额款项；每月可以提现两次；
6. 如有问题，可以致电021-22011427。

三、车主资格

1. 拥有自有的车辆,车况良好；
2. 拥有有效的车辆证明；
3. 持有有效驾照；
4. 3年内无酒后驾驶、危险驾驶、吊销驾照记录。

四、车主责任

1. 车主需要按照与乘客约定的上下车地点和时间准时接送乘客；请乘客尽量设置方便车主停靠的地点；车主可以拒绝停靠交通规则禁止的地点；
2. 车辆规范：车主需要驾驶经过认证的车辆，并保持车内整洁干净，无杂物；
3. 载客人数：车主需要至少保持至少3个空座，供乘客乘坐；不得超载；
4. 行李放置：车主需要保持车辆可以放置至少一个标准行李箱；乘客如有超大行李，需要提前说明；
5. 文明驾驶，遵守交通规则。

五、双方共同遵守的规则

1. 守时规则：车主和乘客均需按照订单时间前后十分钟完成搭乘；如有特殊情况，需要提前电话沟通；
2. 文明拼车：双方均需遵守文明规范，共建和谐社会；
3. 本平台倡议无烟行车；如果有宠物搭乘，需要提前说明；车主有权拒绝未经申请的宠物搭乘；
4. 违反本规则的车主和乘客，平台有权取消注册会员资格，并追究可能导致的损失。
5. 真实拼车，拒绝舞弊行为：88共享出行将采取技术手段监控运营数据，平台有权按照自己的判断标准取消舞弊车主或用户的拼车补贴，对于情节严重者暂停接单2周，甚至取消平台资格。其中作弊订单是指实际未发生拼车行为。亲友间使用本软件拼车，取消订单补贴。所有作弊订单将不予结算，88共享出行有权追究其法律责任。详情请参考拼车公约。

六、取消及退款

1. 原则上车主和乘客均不能取消订单；
2. 确认搭乘前，乘客如行程有变，可以与车主沟通一致后，自助取消订单。确认搭乘后，不得取消；
3. 车主或乘客如有特殊情况，请致电客服申请协商处理；
4. 车主未能按照本规则服务，乘客可以在12小时内致电客服申请退款。

七、88共享出行平台服务

1. 我们为拼车服务提供支付中介服务，确保拼车双方可以顺畅便捷地分摊拼车费用合乘出行；
2. 我们为每次行程提供20万起的拼车保险（车主乘客分摊保险费，平台代收），为拼车双方护航全程；
3. 我们为拼车服务提供支付中介服务，确保拼车双方可以顺畅便捷地分摊拼车费用；
4. 由上汽工会提供保障，为乘客和车主双方护航全程。
