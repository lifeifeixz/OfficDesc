常见问题-车主篇

Q : 为什么加入88共享出行拼车？
A : 88共享出行拼车是基于上下班顺路的原则，为车主和乘客搭建的信息服务平台。成为认证车主，您可以每天搭乘顺路的朋友，与人方便，爱心满满。顺路乘客还可以帮您分摊油费，停车费，减少养车压力。如果足够勤劳的话还可能迅速换新车哟~~同样，在爱车限牌期间，您也可以搭乘其它顺路车上下班。88共享出行拼车根据政府政策搭建本平台，为车主提供合法、安全、公平、健康的平台服务，让你拼车无忧。

Q : 88共享出行拼车是合法的吗？
A：在88共享出行平台拼车完全合法依规。拼车属合乘范畴，88共享出行平台完全符合上海市《关于规范上海市私人小客车合乘出行的实施意见》的相关规定，是上海市唯一备案的合法平台。

Q：在88共享出行平台拼车是否需要修改私家车的运营性质？
A：不需要，88共享出行拼车属于合乘范畴，不属于网约车，不需要更改私家车的运营性质

Q : 在88共享出行拼车接送乘客，双方安全如何保障？
A : 每次使用平台达成的合乘出行的路程和时间内，拼车平台代收订单金额的3%，最低0.2元（费率调整以平台通知为准）用于驾乘人员保险费。此外，我们为拼车双方提供实时的客户服务与GPS全程监控，确保路线安全正确。安心拼车，一路欢笑，保障无忧。

Q : 拼车过程中出现意外，如何处理？
A : 如您在拼车活动中发生交通事故，可直接拨打客服电话：021-22011427。

Q : 提现未到账如何处理？
A：提现申请目前随时都可以发起，平台打款时间为每月的5日与20日（若遇休息日往后顺延），请做好计划提前发起申请。如超时仍未收到，您可拨打客服电话：021-22011427。

Q : 搭乘模式与接送模式的区别？
A： 拼车功能分成两种模式：搭乘模式与接送模式。
搭乘模式：司机发布行程时可添加路过的站点，乘客按时到自选的站点等候上车，乘客无需发布行程。
接送模式：司机与乘客分别发布自己的行程，乘客/司机搭配各自合适的司机/乘客，与对方约定好上车时间、地点。

Q : 车主如何取消订单？
A： 搭乘模式：司机取消订单，不罚款，全额退回乘客（司机24小时只能取消2次）；司机24小时之内取消过2次订单，则无法发布行程。 接送模式：
a. 司机成功接单5分钟内，该订单可取消，不罚款；
b．出发前2小时，司机订单可取消，不罚款，若2小时之内取消订单，将启动协商取消，乘客不接受协商取消，将罚款1元（针对距离出发时间超过2小时的订单）；
c. 若司机有临时状况提出取消订单，将启动协商取消，乘客不接受协商取消，将罚款1元。

Q : 车辆信息有误，如何变更？
A：进入个人中心“我的车”，删除信息有误的车辆，重新添加车辆信息。

Q : 送达乘客后，订单费用何时到账？
A：当天完成的订单费用正常在次日中午12点前到达您的88共享出行“余额”中。

Q : 送达乘客后，对方仍未确认如何处理？
A：建议提醒乘客在到达后进行订单支付，如乘客未支付，请及时与乘客联系，或拨打客服电话：021-22011427。

Q : 如何成为认证车主？
A：注册司机身份则需要完善实名认证信息、驾照认证信息以及车辆认证信息（车牌号中字母要大写），一般88共享出行平台将在2-5个工作日完成认证审核。

Q : 完成认证后，如何接单？
A：系统会根据您设置的出发地和目的地自动匹配推送顺路订单，您可根据实际情况选择订单，抢单成功后建议与乘客沟通联系。

Q : 乘客迟到、爽约如何处理？
A：请您在接单后及时与乘客联系，沟通关于出发时间、地址等问题，如乘客有事无法按时搭乘，双方协商处理即可，如长时间联系不到乘客，您可在出发时间15分钟后短信留言给对方即可离开，感谢您的理解和支持
