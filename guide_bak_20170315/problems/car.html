车主篇

Q : 为什么加入88共享出行拼车？
A : 88共享出行拼车是基于上下班顺路的原则，为车主和乘客搭建的信息服务平台。成为认证车主，您可以每天搭乘顺路的朋友，与人方便，爱心满满。顺路乘客还可以帮您分摊油费，停车费，减少养车压力。如果足够勤劳的话还可能迅速换新车哟~~同样，在爱车限牌期间，您也可以搭乘其它顺路车上下班。88共享出行拼车根据政府政策搭建本平台，为车主提供合法、安全、公平、健康的平台服务，让你拼车无忧。

Q : 88共享出行拼车是合法的吗？
A：交通运输部于2016年7月28日公布了《网络预约出租汽车经营服务管理暂行办法》，其中第三十八条规定：“私人小客车合乘，也称为拼车、顺风车，按城市人民政府有关规定执行”。该政府文件明确了属于互助出行的88共享出行拼车不是网约车经营服务（快车、专车则属于网约车经营服务），拼车不受《网络预约出租汽车经营服务管理暂行办法》的约束。适用于快车和专车的网约车平台、车辆和司机的准入资质条件和运营规范，不适用于88共享出行拼车。

国务院办公厅于2016年7月28日颁布了《关于深化改革推进出租汽车行业健康发展的指导意见》，其中第（十）项明确规定：“私人小客车合乘，也称为拼车、顺风车，是由合乘服务提供者事先发布出行信息，出行线路相同的人选择乘坐合乘服务提供者的小客车、分摊部分出行成本或免费互助的共享出行方式。私人小客车合乘有利于缓解交通拥堵和减少空气污染，城市人民政府应鼓励并规范其发展，制定相应规定，明确合乘服务提供者、合乘者及合乘信息服务平台等三方的权利和义务”。该指导意见明确指出分摊出行成本的拼车有别于经营性网约车，城市人民政府应鼓励并规范拼车共享出行方式。交通部和国务院办公厅颁布的政府文件，是从国家层面对88共享出行拼车平台以及用户拼车行为合法合规的确认！

Q : 车主如何取消订单？

Q : 车辆信息有误，如何变更？
A：进入“我的车”，点击需要修改的车辆进行修改，修改后平台将重新审核车辆信息。

Q : 送达乘客后，订单费用何时到账？

Q : 送达乘客后，对方仍未确认如何处理？

Q : 如何成为认证车主？
A：车主在按要求提交认证材料后，一般88共享出行平台将在2-5个工作日完成认证审核。

Q : 完成认证后，如何接单？
A：系统会根据您设置的出发地和目的地自动匹配推送顺路订单，您可根据实际情况选择订单，抢单成功后建议与乘客沟通联系。

Q : 乘客迟到、爽约如何处理？
A：请您在接单后及时与乘客联系，沟通关于出发时间、地址等问题，如乘客有事无法按时搭乘，双方协商处理即可，如长时间联系不到乘客，您可在出发时间15分钟后短信留言给对方即可离开，感谢您的理解和支持
